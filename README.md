# Ryan B. Dorona Portfolio

Welcome to my personal portfolio website. It highlights my work as a full-stack web developer, including projects built with Laravel, React.js, PHP, MySQL, and modern web technologies.

**Live site:** [ryanbdorona.github.io](https://ryanbdorona.github.io/)

## Navigate the portfolio

- **About** — Learn about my development focus and background.
- **Projects** — Explore selected work, including MANHIS MIS, YUNHAverse Philippines, and my private Cornersteel cooperative loaning-system internship project.
- **Skills** — View the backend, frontend, tools, and technologies I work with.
- **Certifications** — See relevant training and credentials.
- **Contact** — Find my email, phone number, and GitHub profile.
- **Resume** — Opens my current CV in a new tab.

Project cards link to detail pages with an overview, role, technology stack, and available screenshots. Some project previews are intentionally unavailable because they contain private company or member information.

## Run locally

This is a static HTML, CSS, and JavaScript site. No build step or framework is required.

1. Clone or download this repository.
2. Open `index.html` in a browser, or use a local static-server extension for live reload.

## Project structure

```text
assets/
  css/       Site styles
  files/     Resume PDF
  img/       Portfolio images and project previews
  js/        Navigation, carousel, and reveal interactions
projects/    Individual project detail pages
index.html   Portfolio homepage
```

## Deployment

The site is deployed through GitHub Pages from the `main` branch. After pushing updates, allow the GitHub Pages workflow to complete before checking the live site.
